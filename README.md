# sentiment-analysis-project
## 📌 Sentiment Analysis of Movie Reviews
## 📖 Project Overview

This project builds a machine learning model to classify movie reviews as positive (1) or negative (0) using Natural Language Processing techniques.
## 🎯 Objective

To automatically determine the sentiment of movie reviews using text data.
## 📊 Dataset
Source: Labeled movie review dataset
#Columns:
#text: Movie review content
#label: Sentiment (0 = Negative, 1 = Positive)
## ⚙️ Steps Performed
Text cleaning (lowercasing, removing punctuation, stopwords)
Feature extraction using TF-IDF
Model training using Logistic Regression
Model evaluation using confusion matrix
## 🤖 Model Used
Logistic Regression
## 📈 Results
Accuracy: 85.1%
Precision: ~85%
Recall: ~85%
## 🧠 Key Learnings
Text preprocessing is critical in NLP
TF-IDF is a powerful baseline method
Real-world text contains mixed sentiment and noise
## 🛠️ Tech Stack

Pandas
Scikit-learn
NLTK
